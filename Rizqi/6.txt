6. Jelaskan mengenai command git yang sudah pernah kamu coba!
jawab :
- git init = dilakukan untuk pertama kali saat menggunakan git (inisialisasi git)
- git add = perintah untuk menambahkan perubahan ke index / stagged
- git remote = untuk mengkoneksikan dari local ke repo
- git commit = untuk pemaketan dengan keterangan/penjelasan maksud dari perubahan tersebut, ini dilakukan sebelum melakukan push
- git push = melakukan upload perubahan ke repository
- git pull = melakukan download perubahan dari repo
- git merge = penggabungan branch
- git reset = membatalkan perubahan apabila sudah stagged
- git checkout [branch] = berpindah cabang
- git checkout . = membatalkan semua perubahan sebelum di stagged
